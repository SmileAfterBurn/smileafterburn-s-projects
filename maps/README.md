# Карти послуг для постраждалих від ГЗН

Ця папка містить три GeoJSON-файли з переліком ключових послуг у Миколаївській, Одеській та Херсонській областях. Формат GeoJSON легко імпортується у більшість картографічних сервісів (Google My Maps, QGIS, Mapbox Studio тощо).

## Файли карти
- `mykolaiv_services_map.geojson` – ресурси Миколаївської області.
- `odesa_services_map.geojson` – ресурси Одеської області.
- `kherson_services_map.geojson` – ресурси Херсонської області.

Кожен об'єкт містить властивості `category`, `services`, `contacts` та `notes`, що допомагають швидко зорієнтуватися у типі допомоги та способах звернення.

## Як відкрити карту

### Варіант 1. Google My Maps (через браузер)
1. Перейдіть на [https://www.google.com/maps/d/](https://www.google.com/maps/d/).
2. Натисніть кнопку **"Створити нову карту"**.
3. У лівій панелі натисніть **"Імпорт"** та завантажте відповідний `.geojson` файл з цієї папки.
4. Сервіс автоматично відобразить всі точки. Ви можете змінювати стиль шарів, додавати описи та ділитися посиланням з командою.

### Варіант 2. QGIS (настільний застосунок)
1. Встановіть QGIS з [https://qgis.org](https://qgis.org) (якщо ще не встановлено).
2. Відкрийте програму та оберіть **Layer → Add Layer → Add Vector Layer...**.
3. У полі **Source Type** виберіть *File*, натисніть **Browse** та вкажіть шлях до потрібного `.geojson` файлу.
4. Натисніть **Add** – шар з даними з'явиться на карті. За потреби додайте базову підкладку через плагін QuickMapServices.

### Варіант 3. Відображення у браузері через geojson.io
1. Перейдіть на [https://geojson.io](https://geojson.io).
2. Перетягніть файл `.geojson` у вікно браузера або оберіть **Open → File**.
3. Редагуйте й діліться картою через посилання.

## Примітки щодо безпеки
- Координати притулків подані узагальнено (центри громад або офіційні установи), щоби уникнути розкриття точних адрес.
- Перед передачею інформації постраждалим перевіряйте актуальність контактів через гарячі лінії партнерських організацій.
